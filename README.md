# EnergiKaba
Energikaba explores solar-powered mobility solutions for electric rickshaws in Havana, Cuba, focusing on decentralized charging infrastructure to support resilient and sustainable urban transport systems.
Energikaba is a project exploring solar-powered mobility solutions for electric rickshaws in Havana, Cuba.

Many urban transport systems depend heavily on unstable electricity grids. Energikaba investigates how decentralized solar charging infrastructure could improve resilience and accessibility.

The project is in early development and focuses on practical, scalable solutions for real-world conditions.
